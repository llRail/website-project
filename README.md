# Cybersecurity Quiz Web Application

This project is a full-stack web application developed as part of my Computer Science degree at Brunel University.

## Overview
The application helps users learn cybersecurity concepts through an interactive quiz system. It tracks progress and adapts difficulty based on performance.

## Features
- Interactive quiz with multiple question types
- Timer and progress tracking
- Adaptive difficulty system
- Review mode with feedback on answers
- Study material unlocking based on points

## Technologies Used
- Frontend: React.js, JavaScript, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- APIs: RESTful services (JSON/HTTP)

## My Contribution
I developed the application end-to-end, including:
- Building the React frontend
- Creating REST APIs with Node.js and Express
- Managing data with MongoDB
- Implementing scoring, timers, and user progress tracking
